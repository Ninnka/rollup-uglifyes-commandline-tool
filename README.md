# rollup-uglifyes-commandline-tool

> rollup-uglifyes-commandline-tool is a command-line tool for uglifing javascript(es6)

## usage

```bash
# install
npm i
# global
npm link
```

## note

- default node path:

```bash
/bin/env node
```

- if you need to change node path, modify the first line in bin/rpes.js
